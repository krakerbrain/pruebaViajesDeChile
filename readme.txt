Prueba Viajes de Chile
- La versión de Boostrap usada fue la 4.6.

- Se utilizaron todas las etiquetas semánticas para estructurar el HTML. La página tiene definida la barra de navegación, el header, 4 sections, una de ellas usada para el formulario de contacto de Typeform, y el footer.

- Se usaron 7 iconos de fontawesome (version 5.0) Se usaron muchos componentes y estilos de Boostrap, sin embargo, en algunas partes se uso CSS para darle mejor y más practico estilo a la pagina. La fuente usada en el proyecto fue Roboto 300 y 500. Boostrap define los títulos con la etiquete H1 por lo cual se le aplicó el weight 500 y a los p (parrafos) weight 300.
